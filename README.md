AJ Multi Assistant

Monorepo scaffold with `frontend/` (React + Firebase Auth) and `backend/` (Node + Express + Mongo + Firebase Admin).

Quick start (each service):

- Frontend
  - cd frontend
  - npm install
  - npm run dev

- Backend
  - cd backend
  - npm install
  - copy `.env.example` to `.env` and fill values
  - npm start

There's a `docker-compose.yml` for local development (Mongo + backend + frontend).
